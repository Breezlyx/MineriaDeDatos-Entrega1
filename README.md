# Proyecto: Minería de Datos - Análisis de Salarios

## Justificación del Conjunto de Datos

Para el desarrollo de este proyecto a lo largo del semestre, hemos seleccionado el **"Job Salary Prediction Dataset"**. 

* **Fuente:** [Kaggle - Job Salary Prediction Dataset](https://www.kaggle.com/datasets/nalisha/job-salary-prediction-dataset)

La elección de este conjunto de datos se fundamenta en los siguientes criterios técnicos y académicos exigidos para la asignatura:

1. **Estructura Mixta de Variables:** El dataset cumple con el requisito fundamental de contener una rica variedad de tipos de datos. Posee variables **numéricas** clave como el salario (`salary`), años de experiencia (`experience_years`) y cantidad de habilidades (`skills_count`); así como variables **categóricas** como la industria (`industry`), tamaño de la empresa (`company_size`), ubicación (`location`) y modalidad de trabajo (`remote_work`).
2. **Volumen y Amplitud Adecuada:** Cuenta con una cantidad extensa de registros, lo que garantiza la viabilidad y robustez estadística para las siguientes etapas del proyecto. Este volumen evitará el problema de contar con un conjunto acotado que requiera buscar más datos en el futuro.
3. **Escalabilidad para Modelos de Machine Learning:** La estructura del dataset nos permite darle continuidad durante todo el semestre. Presenta escenarios claros tanto para aplicar **modelos de regresión** (por ejemplo, predecir el salario exacto de un profesional basado en su experiencia) como para **modelos de clasificación** (por ejemplo, predecir la modalidad de trabajo o el tamaño de la empresa).
4. **Relevancia del Dominio:** El análisis de tendencias salariales, habilidades tecnológicas y modalidades de trabajo es un tema socioeconómico de alto interés actual, lo que facilita enormemente la etapa de comprensión del negocio y la posterior interpretación de los resultados (Fase inicial de KDD/CRISP-DM).
